# Code for Device Paper

This repository contains the code used in the paper:

**Modular Tactile Sensor Platform with Sequential Optical Sampling for Scalable Human–Robot Interfaces**  
*Device* (Cell Press), Manuscript ID: DEVICE-D-25-00477R1

## Contents
- `ver3_5fold2_4input_Super_resolution_DEVICE_final.ipynb`  
  - Main notebook for training and evaluation  
  - Reproduces Figures 3–5 in the main text

## Requirements
- Python 3.9
- PyTorch 2.x
- CUDA 11.x

## Notes
- This repository is a frozen version of the code corresponding to the published paper.
- Paths and dataset locations may need to be adjusted by the user.
